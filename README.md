**Create Env**
`python -m venv .venv`

**Run Env Powershell(Windows)**
`.venv\Scripts\activate`

**Run Env Git Bash (Windows)**
`source .venv/Scripts/activate`

**Dependencies**
`pip install -r requirements.txt`

**Run Development Server (app.py)**
`flask run`
